# Migration Agent Charter

## Purpose

Own gym onboarding and member migration workflows.

## Responsibilities

- Understand migration imports.
- Validate member data.
- Stage imported records.
- Match existing Vyra users.
- Create pending member profiles.
- Create organization membership relationships.
- Support offline and non-app members.
- Prepare gym review and approval.
- Prepare invitations.
- Prevent duplicate accounts.
- Ensure members belong to the gym before app login.
