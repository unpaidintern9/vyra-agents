# Product Feedback

Product feedback records capture source, category, frequency, severity, linked features, linked customers, and recommendations.

Supported sources include coaches, gyms, athletes, Customer Success, Sales, Marketing, and Executive.

Command:

- `npm run engineering:feedback`

Feedback can inform feature prioritization, release communication readiness, roadmap risks, and customer-facing support planning. It does not send messages, update customer accounts, publish release notes, create GitHub issues, or start engineering work automatically.
