# Pricing Library

The pricing library stores local reference prices for Vyra Performance products and services.

Supported references include athlete plans, coach plans, gym software plans, enterprise references, white-label references, setup services, and migration services.

Pricing records are versioned and audit-friendly. They can be used in local reports, forecasts, proposal prep context, Sales planning, Marketing positioning review, and Executive summaries.

The library does not update Stripe, invoices, billing systems, customer accounts, or external pricing pages.
