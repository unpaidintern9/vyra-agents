# Cross-Agent Metrics

Cross-agent metrics track local operating signals across agents.

Tracked metrics include:

- task completion rate
- overdue task count
- blocked task count
- workflow aging
- approval backlog
- customer onboarding completion
- sales pipeline health
- campaign readiness
- content approval backlog
- revenue forecast confidence
- engineering release readiness
- goal progress

Metrics are generated from local records and shared queues. They do not install tracking scripts, sync analytics tools, collect customer data, or trigger autonomous external actions.
