# Knowledge Library

The Knowledge Library stores structured records separate from file assets.

Supported records include FAQ, SOP, Best Practice, Process, Playbook, Internal Guide, Troubleshooting, Product Documentation, Release Notes, and Training Material.

Each record stores title, summary, category, linked assets, linked products, linked audiences, linked tasks, linked workflows, confidence, approval status, version, usage references, and audit history.

Knowledge records integrate with Shared Memory, Universal Tasks, Executive Planning, Marketing, Marketing Content Studio, and Sales through local IDs and references.

Use `npm run assets:knowledge` to inspect the local knowledge library.
