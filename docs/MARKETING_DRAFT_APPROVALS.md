# Marketing Draft Approvals

Marketing draft approvals are local review records for Content Studio drafts.

Supported draft statuses:

- draft
- needs_review
- approved
- rejected
- archived

Supported transitions:

- submit for review
- request edits
- approve draft
- reject draft
- archive draft

Every transition records timestamp, previous status, new status, reviewer or operator, reason, and audit event.

Approval is internal only. It does not publish content, post to social, send email, buy ads, sync CRM data, or create any external action.
