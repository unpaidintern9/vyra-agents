# Evidence Library

The evidence library links proposal claims to local, auditable sources.

Evidence items contain:

- ID
- type
- description
- source
- linked requirement
- linked proposal
- linked fact
- linked report
- confidence
- verification status
- audit history

Evidence may point to Local CRM records, research intake, workflow handoffs, shared memory facts, reports, and task records.

The evidence library does not browse, scrape, enrich externally, or collect restricted data. Missing evidence remains visible for Operator and Proposal Prep review.
