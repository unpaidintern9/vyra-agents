# Integrations

Phase 3 prepares read-only integration shapes for the Vyra Agent Platform.

## Current Status

- GitHub: mock/readiness only.
- Supabase: mock/readiness only from the dashboard.
- Stripe: planned.
- Apple App Store: planned.
- Google Play: planned.
- SendGrid: planned.
- Slack: planned.
- Discord: planned.
- Sentry: planned.
- Apple Health: planned.

The dashboard does not call production APIs, write production data, or use AI.

## GitHub

The GitHub integration foundation models repository name, remote URL, default branch, latest commit, PR count, issue count, workflow status, last updated, and health status.

Real GitHub API checks come later as read-only integrations.

## Supabase

The Supabase integration foundation models project status, environment, migration count, latest migration, database, auth, storage, edge functions, last checked, and health status.

The dashboard does not connect to production Supabase yet.
