# Compliance Matrix

The compliance matrix is a deterministic local tracker for proposal requirements.

Each requirement stores:

- requirement ID
- proposal
- source
- section
- requirement text
- status
- evidence
- reviewer
- confidence
- notes
- risks
- missing items

Supported statuses:

- Unknown
- Pending
- Partially Addressed
- Addressed
- Needs Review
- Approved

Compliance approval is never automatic. A requirement may be marked addressed from local evidence, but approval requires a human reviewer.
