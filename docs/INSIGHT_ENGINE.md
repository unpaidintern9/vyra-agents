# Insight Engine

The Insight Engine creates deterministic local insights for growth risk, stalled sales opportunities, campaign readiness gaps, churn exposure, forecast risk, engineering bottlenecks, blocked task clusters, workflow aging, overdue executive decisions, goal slippage, missing data, and weak confidence areas.

Each insight includes:

- insight ID
- title
- category
- severity
- confidence
- explanation
- linked entities
- linked tasks
- linked workflows
- recommended next action
- owner agent
- status

Statuses are `new`, `acknowledged`, `assigned`, `in_review`, `resolved`, and `archived`.

Insights are advisory and do not execute external actions.
