# Roadmap Management

Roadmap records support quarterly, monthly, release, and long-term planning.

Each roadmap stores milestones, features, goals, dependencies, risks, progress, executive priority, and audit history.

Commands:

- `npm run engineering:roadmaps`
- `npm run engineering:roadmap-report`

Roadmap reports are generated under:

```text
codex-agent-threads/shared/engineering/reports/
```

Roadmaps are advisory planning artifacts. They do not create tasks automatically, approve work, write to GitHub, or change release status.
