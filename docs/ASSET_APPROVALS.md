# Asset Approvals

Asset approvals support Draft, Under Review, Approved, Deprecated, and Archived states.

Approval records store reviewer, approval date, notes, and audit history. Approval only marks a local asset reference as approved for internal use. It does not publish, upload, email, post, sync, or distribute anything externally.

Use:

- `npm run assets:approve`
- `npm run assets:archive`
- `npm run assets:validate`

Automatic asset approval is disabled.
