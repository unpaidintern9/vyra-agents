# Sales Research Workflow

The Sales Agent research workflow starts with local prospect slots and manual/public fields.

## Workflow

1. Select or search for a prospect in the Sales dashboard.
2. Add known website, social, contact, location, software, member-count, and program notes through local intake or manual research mode.
3. Run local enrichment to identify business type, likely pain points, ICP fit, recommended Vyra product, outreach angle, missing info, and next actions.
4. Export a dossier or create local follow-up tasks before any manual outreach.

## Manual Research Mode

Paste only public, operator-reviewed information:

- Website URL
- Public social URL
- Public contact page URL
- Publicly listed owner/operator name
- Current software clues
- Schedule or program notes
- Public member-count proxy

Do not paste private data, restricted pages, login-gated content, paywalled data, secrets, or sensitive personal data.
