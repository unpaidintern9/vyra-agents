# Agent Memory Views

Phase 52 filters shared memory for each agent.

## Sales

Shows organizations, contacts, opportunities, Sales workflows, proposal readiness, relationship health, and research facts.

## Executive

Shows strategic entities, high-value opportunities, approval items, risks, summaries, cross-agent activity, and decision history.

## Operator

Shows assigned tasks, missing data, verification needs, blocked items, handoffs, duplicate queues, and stale facts.

## Proposal Prep

Shows proposal-ready opportunities, missing requirements, approved sources, decision makers, readiness facts, and required artifacts.

## Contract Intelligence

Shows contract opportunities, NAICS context, set-aside relevance placeholders, compliance facts, proposal dependencies, and organization links.

## Marketing Placeholder

Prepares local structure for brand assets, campaign ideas, approved messaging, audience segments, and marketing facts. Phase 52 does not build a full Marketing Agent.
