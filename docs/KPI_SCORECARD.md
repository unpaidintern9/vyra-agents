# KPI Scorecard

Executive KPI records include KPI ID, name, description, category, current value, target value, unit, trend, status, linked goal, linked tasks, linked reports, confidence, last updated timestamp, and audit history.

KPI evaluation is deterministic:

- ahead
- on_track
- behind
- at_risk
- blocked
- unknown

KPI status is advisory and never triggers external action. Linked blocked goals can mark a KPI as blocked until manual review clears the goal blocker.
