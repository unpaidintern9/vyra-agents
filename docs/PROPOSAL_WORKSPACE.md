# Proposal Workspace

Proposal workspaces are local planning records linked to existing opportunities, workflows, tasks, shared memory, research, reports, and Executive planning goals.

Each workspace tracks:

- identity, owner, contributors, status, and priority
- due dates, review dates, and planned submission date
- related workflow, task, research, report, and shared memory references
- readiness scorecard
- compliance matrix
- section progress
- evidence library
- review queue
- immutable timeline

Dashboard surfaces:

- Sales: Proposal Workspace, Proposal Pipeline, Proposal Readiness, dependencies, risks, compliance, evidence, and section progress.
- Executive: Proposal Portfolio, Proposal Health, Executive Review Queue, Compliance Risks, Upcoming Deadlines, and High-Risk Proposals.
- Operator: Proposal Task Queue, Missing Sections, Missing Evidence, Compliance Gaps, Review Assignments, and Blocked Proposal Tasks.

The workspace is advisory. It prepares work for human review; it does not submit, send, approve, browse, or sync anything externally.
