# Goal Management

Executive goals include owner, user, category, priority, status, timeframe, start date, target date, linked tasks, workflows, entities, opportunities, organizations, reports, KPIs, blockers, risks, decisions, progress score, confidence score, and audit history.

Supported statuses:

- draft
- active
- at_risk
- blocked
- review
- completed
- paused
- archived

Every status transition records timestamp, previous status, new status, actor, reason, affected tasks, affected workflows, and next action.

Goal completion is never automatic unless deterministic evidence is added and a valid local transition is recorded.
